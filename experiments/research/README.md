# Research Mirrors

This directory contains curated public mirrors of technical report experiment folders
from the private `Banterhearts/research` workspace.

Included:
- source code
- configs
- run scripts
- Dockerfiles
- lightweight documentation

Excluded:
- generated `results/` trees
- caches and `__pycache__/`
- local model artifacts
- notebook exports and other bulky derived outputs

Canonical publish-ready reports live under `outputs/publish_ready/reports/`.
