# Publish-Ready Supporting Docs

This directory contains publication-quality supporting documents that are not
part of the numbered technical report archive.

## What Lives Here

- benchmark deep dives such as `Gemma3_Benchmark_Report.md`
- long-form supporting writeups such as `Performance_Deep_Dive.md`
- artifact/reproduction notes such as `TR117/README.md`

## What Does Not Live Here

- numbered TR-series reports
- conclusive synthesis volumes
- executive whitepapers

Those belong in `../reports/`.

## Guidance

- Link `../reports/` when you need a canonical technical report citation.
- Link this directory when you need benchmark-specific supporting context.
